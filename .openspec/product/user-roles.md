# User Roles

## customer
Cliente final.

Permisos:
- ver meals
- crear pedidos
- ver progreso
- gestionar perfil

## nutritionist
Profesional nutricional.

Permisos:
- gestionar pacientes
- crear planes
- gestionar turnos

## kitchen
Operaciones internas.

Permisos:
- gestionar stock
- actualizar pedidos
- visualizar comandas

## admin
Administrador global.

Permisos:
- acceso total
- analytics
- usuarios
- métricas
